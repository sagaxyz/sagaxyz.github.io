# cluster

The cluster command helps you launch a cluster onto a kube services provider.

Currently, we only support aws, and the designated cluster name is saga-eks.

$ saga cluster 
Cluster-related commands

Usage:
  saga cluster [command]

Available Commands:
  delete      Deletes an EKS cluster from a host. Currently only supports aws
  deploy      Deploys an EKS cluster to a host. Currently only supports aws
  list        Lists the existing clusters

Flags:
  -h, --help   help for cluster

Global Flags:
  -l, --loglevel string   Log level (default "info")

Use "saga cluster [command] --help" for more information about a command.

The cluster command uses the current configuration with aws cli. At this moment, we only support v2 of aws-cli. See installing (opens new window) and upgrading (opens new window) from v1.

# Commonly Used Commands

saga cluster deploy aws - Creates and deploys a cluster to your AWS account

saga cluster delete aws - Deletes the created cluster and all associated infrastructure components
